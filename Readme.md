## Usage
**Production**
`npm i`
`cd front/remote-control`
`npm i`
`npm run start:all`
* App served back-end
* WebSocket URL: `ws://localhost:5000`
* * App served front-end
* WebSocket URL: `ws://localhost:8181`

**Developer**
`npm run start:dev`
* App served back-end
* WebSocket URL: `ws://localhost:4000`
